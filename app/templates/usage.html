Usage

Add this at the bottom of your postgresql.conf file:

include = 'tuning.conf'

Select your profile and create the file tuning.conf inside your PGDATA directory and add this content:

  • {{ env.name }} 
## Generated by PGConfig {{ version }}
{{ formatParameters(env.name) }}
## Connection stuff
max_connections = {{ form.max_connections }}
listen_addresses = '*'
