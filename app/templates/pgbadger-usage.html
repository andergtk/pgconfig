PGBadger Configuration

Add this at the bottom of your postgresql.conf file:

include = 'pgbadger.conf'

Select your log manager and create the file pgbadger.conf inside your PGDATA directory and add this content:

  • {{ log_format }} 
## Generated by PGConfig {{ version }}
# Using '{{log_format}}' log format

{{ formatBadgerSpecifics(log_format) }}
log_checkpoints = on
log_connections = on
log_disconnections = on
log_lock_waits = on
log_temp_files = 0

lc_messages = 'C'

RSysLog Configuration

Add this at the bottom of your /etc/rsyslog.conf file:

local0.*         -/var/log/pgsql.log

...and make a adjustment on the /var/log/rsyslog.conf file to add a exclusion of local0 on /var/log/messages: 

*.info;mail.none;authpriv.none;cron.none;local0.none                /var/log/messages

Now restart rsyslog service:

service rsyslog restart
